Vectorization Project

CS110 – Python Optimization with NumPy

This project explores the use of vectorization in Python to improve the performance of numerical computations using the NumPy library. Instead of writing slow for-loops, we use vector operations that are faster and more efficient.

In the notebook, we compared loop-based approaches to vectorized ones, ran performance benchmarks, and practiced using array broadcasting and slicing.

Example from the project:

import numpy as np

a = np.array([1, 2, 3])
b = np.array([4, 5, 6])
result = a + b
print(result)  # Output: [5 7 9]

This approach avoids looping through each element and instead performs the operation all at once on entire arrays.
