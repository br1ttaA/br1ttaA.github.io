Data Exploration & Visualization

By Britta Anderson

Project Overview

This project focused on exploring and visualizing a real-world dataset using Python. Inspired by a lab on the Titanic dataset, I selected a new dataset of interest, performed exploratory data analysis (EDA), and created visualizations to uncover insights.

Technologies Used

  • Python
  • Pandas
  • Matplotlib
  • Google Colab

1. Dataset Loading and Inspection

The dataset was loaded using pandas and the initial structure was inspected with head():

import pandas as pd

df = pd.read_csv("data.csv")
df.head()

2. Data Cleaning

Missing values were identified and cleaned to ensure accurate analysis:

df.isnull().sum()
df.dropna(inplace=True)

3. Exploratory Data Analysis

Basic statistics and distributions were explored:

df.describe()
df["Category"].value_counts()

4. Data Visualizations

Histograms and bar charts were created using Matplotlib:

import matplotlib.pyplot as plt

plt.hist(df['Age'])
plt.title("Age Distribution")
plt.xlabel("Age")
plt.ylabel("Frequency")
plt.show()
df['Gender'].value_counts().plot(kind='bar')
plt.title("Gender Breakdown")
plt.show()

5. Conclusions

This project improved my data analysis skills, deepened my understanding of pandas and matplotlib, and showed how data can be used to support meaningful insights. I learned how to ask the right questions, clean messy data, and create visual narratives from datasets.
